# k6 Load Testing
 
## Installation
 
Install k6 via Homebrew:
 
```bash
brew install k6
```
 
## Running the Test
 
```bash
k6 run k6/load-test.js
```
 
Make sure your NodeBB instance is running at `http://17313-team15.s3d.cmu.edu:4567` before running the test.
 
## Output
 
Terminal output from a full test run can be found in `k6/output.txt`.
